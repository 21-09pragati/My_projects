# 🎮 Guess The Number Game

A fun and beginner-friendly web game where users try to guess a randomly generated number between 1 and 100.

## 🚀 Technologies Used
- HTML
- CSS
- JavaScript

## ✨ Features
- Input validation and feedback
- 10 chances to guess
- Random number logic
- Restart button
